# Contributing to ArenaFrame

Thank you for your interest in contributing to ArenaFrame! This Detroit-inspired PvP arena project welcomes contributions from the community.

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion:

1. **Check existing issues** to avoid duplicates
2. **Create a new issue** with a clear title and description
3. **Include details**:
   - What you expected to happen
   - What actually happened
   - Steps to reproduce (if applicable)
   - Screenshots or videos (if helpful)

### Suggesting Features

We welcome feature suggestions that:
- Enhance gameplay balance
- Improve the Detroit theme authenticity
- Add value to the player experience
- Maintain the minimalist aesthetic

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/YourFeature`
3. **Make your changes**
4. **Test thoroughly** (especially Verse scripts)
5. **Commit with clear messages**: `git commit -m "Add feature: YourFeature"`
6. **Push to your fork**: `git push origin feature/YourFeature`
7. **Open a Pull Request**

### Code Standards

#### Verse Scripts
- Follow Epic's Verse style guide
- Use meaningful variable names
- Comment complex logic
- Test in UEFN before submitting

#### Documentation
- Use clear, concise language
- Include examples where helpful
- Maintain consistent formatting
- Respect Detroit cultural references

#### Asset Specifications
- Follow existing format in README files
- Include performance targets
- Specify technical requirements
- Maintain Detroit aesthetic guidelines

## Development Setup

### Prerequisites
- Unreal Engine 5.1+ with UEFN
- Git
- Text editor (VS Code recommended)
- Basic Verse knowledge

### Local Setup
```bash
git clone https://github.com/Balentine-Tech-Solutions/ArenaFrame.git
cd ArenaFrame
# Open in UEFN
```

## Testing

Before submitting:
- [ ] Verse scripts compile without errors
- [ ] No performance regressions
- [ ] Documentation is updated
- [ ] Changes align with Detroit theme
- [ ] Code follows style guidelines
